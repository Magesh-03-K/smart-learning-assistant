# Smart Learning Assistant 🚀

An AI-powered learning roadmap generator that creates personalized study plans with YouTube resources.

## 🌍 Live Demo
https://smart-learning-assistant-seven.vercel.app/

---

## ✨ Features

- AI-generated learning roadmaps
- Dynamic day detection
- YouTube tutorial integration
- Progress tracking
- Local storage persistence
- Responsive modern UI
- Full-stack deployment

---

## 🛠 Tech Stack

### Frontend
- React
- CSS

### Backend
- Node.js
- Express

### APIs
- Gemini AI API
- YouTube Data API

### Deployment
- Vercel
- Render
